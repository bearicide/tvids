Backend-ready upload notes
Static GitHub Pages cannot permanently accept public uploads by itself. This build keeps files in the visitor browser session with objectURL previews. Later backend targets: Cloudflare R2, Firebase Storage, Supabase Storage, or a small upload API. Hook point: handleFiles(files).
Recommended next step: add moderation queue, file size limit, accepted MIME allowlist, thumbnail generation, and public gallery JSON.
